# NTUMC WordNet Tagger

A system for tagging corpus text with WordNet synsets.

## Overview

The NTUMC WordNet Tagger is a tool for automatically connecting words in a corpus with their corresponding WordNet synsets. It supports multiple languages and can handle both single-word expressions (SWEs) and multi-word expressions (MWEs).

## Supported Languages

- English (en)
- Czech (ces)
- Japanese (jap)
- Mandarin Chinese (zsm)
- Vietnamese (vie)

## Features

- Automated corpus tagging with WordNet synsets
- Support for single-word and multi-word expressions
- Language-specific tagging rules
- Configurable tagging parameters
- Database integration with WordNet and corpus databases
